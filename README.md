A collection of assignments and project work completed during Machine Learning sessions at GLA University.

This repository will be continuously updated throughout the course as new topics, assignments, datasets, and experiments are introduced.

---

## 📚 Repository Purpose

The goal of this repository is to:

* Track progress throughout the Machine Learning course
* Maintain organized assignment submissions
* Practice Python-based data analysis and visualization
* Implement Machine Learning concepts learned during sessions
* Build a reference repository for future learning and revision
